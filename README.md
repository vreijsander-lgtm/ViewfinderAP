# ViewfinderAP
A Archipelago randomizer for Viewfinder
(still work in progress)

Options
Progressive camera
on/off

Levels
Progressive: 1-1 then 1-2 etc.
Fully random: 3-3 then 2-5 etc.
off: levels wont be added to the pool

Goals
The last level
All levels
all filters
All levels incl optional lvls

Filters
On/off
